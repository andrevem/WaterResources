A [Scoop](https://scoop.sh) bucket for installing water resources-related software.

## How do I install a manifest from this bucket?
1. Add this bucket: `scoop bucket add https://github.com/andrevem/WaterResources`
2. Install a manifest: `scoop install <manifest> # e.g. scoop install epanet`
